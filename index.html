1. What is Virtual DOM

  • React keeps a lightweight representation of the real DOM in the memory, and that is known as the virtual DOM.
  • When the state of an object changes, the virtual DOM changes only that object in the real DOM, rather than updating all the objects.

2. What is SPA ?

A single-page application(SPA) is an application that loads a single HTML page and all the necessary assets (such as JavaScript and CSS) required for the application to run.Any interactions with the page or subsequent pages do not require a round trip to the server which means the page is not reloaded.

3:What is difference between class and functional component?

Class Component
  • A class component requires you to extends from React.Component and creates a render function which retirn a react element.
  • It must have the render() method returning the JSX which is syntactically similar to HTML.
  • It also known as "Stateful component" as they implemet logic and state.
  • React lifecycle method can be used inside the class component.
  • Constructor are used in class component as it need to stored the data and initializes the state.
Functional Component
  • A functional component is just a plain javascript function that accepts props as an arguments & returns a react elements.
  • There is no render method used in functional component.
  • It is also known as stateless component as they simply accepts data abd display them in some form.
  • React lifecycle methods cannot be used in the functional component.
  • Constructors are not used in the functional compoenent.


4. What does mean by state and its use in react?

  • State is similar to props, but it is private and fully controlled by the component ,i.e., it is not accessible to any other component till the owner component decides to pass it.
  • it is used to manipulate data within a component.
  • it can be declared within a Constructor or outside constructor.
  • it can be changed using this.setState only.
  • it changes can be asynchronous.

5. What is JSX and why do we use it instead of js?

JSX is popularly used in React, as it makes the job of building an application much easier. JSX is an addition to the JavaScript syntax which is a mixture of both HTML and JavaScript. Both JS and JSX are interchangeable but JSX makes the code easier to understand for users.
